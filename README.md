## Domain
Education

## Objective
To perform end-to-end Exploratory Data Analysis (EDA) on student performance data using Python, with the goal of identifying key factors that influence academic outcomes.

## Dataset
- Source: Kaggle / UCI Machine Learning Repository / Public Dataset
- Records: 500+ student records
- Features: 10+ (Numerical and Categorical)

## Tools & Libraries
- Python
- Google Colab

## Analysis Performed
- Data Loading and Initial Exploration
- Data Cleaning & Preprocessing
- Handling Missing Values and Duplicates
- Feature Engineering
- Univariate, Bivariate & Multivariate Analysis
- Statistical Summaries
- 10+ Meaningful Visualizations

## Key Insights
1. Study hours have a positive impact on final exam performance.
2. Attendance rate shows a strong correlation with student scores.
3. Parental education level influences academic outcomes.
4. Students with internet access perform better on average.
5. Minor performance variations are observed between genders.

## Conclusion
This project highlights important factors affecting student performance. The insights derived can help educators and institutions make data-driven decisions to improve academic planning and student support strategies.
